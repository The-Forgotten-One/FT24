# Электронный журнал студентов

Это статический сайт-журнал для публикации текущей успеваемости студентов.

## Что внутри

- красивый адаптивный интерфейс;
- группы студентов;
- поиск по ФИО;
- сортировка;
- 8 лабораторных работ;
- итоговый балл, процент и статус.

## Где менять данные

- `data/students.js` — группы, студенты и их баллы;
- `data/course.js` — название курса, количество лабораторных, пороги.

## Как запустить

Достаточно открыть `index.html` в браузере.
Никакой сборки не требуется.
